# Style Modes Reference

## Mode 1: Formal Contrast (正经反差)

Serious institutional formats filled with playful AI data. The humor comes from the deadpan gap between the format's authority and the silly content.

**Core tension**: Official format + unofficial data = comedy

**Format examples**:
- Exam paper (密封线, red pen corrections, examiner comments)
- Medical prescription (diagnosis: "severe human-AI dialogue dependency")
- Imperial edict (奉天承运, golden scroll, vermilion seal)
- Bank statement (transaction ledger, running balance, overdraft warnings)
- Report card (three subjects = three AI tools, homeroom teacher comments)
- Court verdict / legal document
- Tax return
- Scientific paper abstract

**Design rules**:
- Maintain the format's visual conventions faithfully — the joke only works if the format looks authentic
- Data fills real fields: dates, amounts, grades, diagnoses
- Tone: deadpan serious. Never wink at the camera.

## Mode 2: Visual Art (视觉美学)

Cards that are gallery-worthy. Strong visual composition, art-directed layouts, emotional color palettes.

**Core quality**: Someone should want to frame this.

**Format examples**:
- Vinyl record (tracks = days, grooves = intensity, center label = total)
- Movie poster (credits = AI tools, rating = daily average, tagline = quote)
- Tarot card (major arcana, elemental symbols for sources, seven-pointed star)
- Star map (stars = days, brightness = turns, nebulae = sources)
- Wine label (Grand Cru, blend ratios = source percentages)
- Music player (Spotify-style, progress bar, playlist)
- Concert ticket (neon, multiple acts = AI tools)

**Design rules**:
- Visual impact first, data legibility second (but data must still be findable)
- Rich, intentional color palette — not generic
- Large negative space is acceptable
- Typography is a design element, not just a carrier

---

## Mode 3: Geek Mode (极客硬核)

Data-dense, precision-obsessed, technically beautiful. For people who think dashboards are art.

**Core quality**: Information density + visual precision.

**Format examples**:
- Terminal CLI (ASCII bars, ANSI colors, monospace everything)
- NASA mission control (telemetry panels, channel status, mission timeline)
- K-line stock chart (candlesticks = daily change, volume bars, analyst notes)
- Apple Watch fitness rings (three rings = three sources, watch frame)
- Cyberpunk ID card (neon cyan + magenta, barcode, threat level)
- Server monitoring dashboard
- Git contribution graph

**Design rules**:
- Dark backgrounds (almost always)
- Monospace or technical fonts dominate
- Every pixel carries data
- Status indicators (green dots, warning triangles) are encouraged
- Glow effects used sparingly for emphasis

---

## Mode 4: Life Remix (生活脑洞)

Everyday objects reimagined as data carriers. Warm, funny, relatable. The kind of card that makes someone say "haha" before "wow."

**Core quality**: "I use this thing every day — but I never saw my AI data in it."

**Format examples**:
- Food delivery order (menu items = AI tools, order total = turns)
- Coffee cup sleeve (round layout, membership tier, cup count)
- Recipe (ingredients = AI tools, cooking steps = daily rhythm)
- IKEA assembly manual (parts diagram, step-by-step, warning icons)
- Receipt / till slip (itemized list, barcode, "thank you come again")
- Fortune cookie (lucky numbers, fortune reading, emoji calendar)
- Express delivery slip (sender → recipient, "warehouse exploded! 166 packages")
- WeChat Moments (nine-grid + comment section)

**Design rules**:
- Warm color palettes (cream, brown, yellow, soft green)
- Rounded, friendly typography
- Small humorous details reward close reading
- Format should be instantly recognizable from daily life

---

## Mode 5: Time Travel (时空穿越)

Cross-era and cross-culture mashups. Old meets new. East meets West. Ancient meets digital.

**Core quality**: Temporal or cultural dislocation — the data is modern, the format is not.

**Format examples**:
- Republican-era newspaper (号外! headline, two-column layout, traditional type)
- Telegram (STOP-delimited, all-caps Chinese numerals, dispatch bureau)
- Passport visa page (three AI sources = three jurisdiction stamps, MRZ code)
- Boarding pass (departure date → arrival date, seat class = user tier)
- Postcard (left image, right letter, stamps and postmarks)
- Manga panel (speed lines, speech bubbles, explosion frames)
- Pokédex entry (red device, green scanner screen, rarity stars)
- Imperial exam golden list (科举金榜, three subjects, status: 状元)

**Design rules**:
- Period-accurate typography and layout conventions
- Use era-appropriate language (classical Chinese, telegraph style, etc.)
- Modern data in antique clothing — the anachronism IS the design
- Aging/texture effects (paper grain, ink bleed) add authenticity
